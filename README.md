# RevoBank Customer Segmentation

## Overview

This project analyzes customer transaction behavior and financial characteristics to identify customer segments that can help increase transaction activity and maximize profitability.

## Objectives

* Clean and prepare customer and card datasets
* Analyze transaction behavior and financial performance
* Identify customer segments using K-Means Clustering
* Generate actionable business recommendations

## Tools

* Python
* Google Colab
* Pandas
* NumPy
* Matplotlib
* Scikit-Learn

## Key Analysis

* Data Cleaning & Validation
* Feature Engineering
* Net Profit Analysis
* Fraud Rate Analysis
* Transaction Behavior Analysis
* Customer Segmentation

## Segmentation Method

* K-Means Clustering
* RobustScaler
* Elbow Method
* Silhouette Analysis

## Project Deliverables

* Data Cleaning Process
* Exploratory Data Analysis (EDA)
* Customer Segmentation

* Final Presentation
